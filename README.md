# Hybrid Metaheuristic ANN Prediction

## Overview

This project compares four hybrid machine learning models for predictive analysis using Artificial Neural Networks and metaheuristic optimization techniques.

The objective is to evaluate the performance of different hybrid approaches and identify the most effective model based on prediction accuracy and error metrics.

## Models Evaluated

* TLBO-ANN (Teaching Learning Based Optimization – ANN)
* SHO-ANN (Spotted Hyena Optimizer – ANN)
* HHO-MLP (Harris Hawk Optimization – MLP)
* SSA-BP (Sparrow Search Algorithm – Back Propagation)

## Technologies Used

## Dataset

The dataset contains more than 10,000 samples and multiple input features used for predictive modeling.

Input Features:

* SLA
* SH
* PWPR

Target Variable:

* FS

## Evaluation Metrics

The models are compared using:

* R2 Score
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
