# Hybrid-metaheuristic-ann-prediction
Comparison of 4 hybrid optimization + neural network models:
- TLBO-ANN (Teaching Learning Based Optimization)
- SHO-ANN (Spotted Hyena Optimizer)
- HHO-MLP (Harris Hawk Optimization)
- SSA-BP (Sparrow Search Algorithm)

## Dataset
10,000 samples | 7 input features | Target: FS (Factor of Safety)
